CSS Border radius

  • It is used to convert sharp edge corners into curved edge corners for the Html elements.

  • It accepts value in pixel format.This property accepts the four values at a time.

  • The four values are as follows,

    1. Represents left-top corner (px)

    2. Represents right-top corner (px)

    3. Represents right-bottom corner (px)

    4. Represents left-bottom corner (px)

  • Note : We can measure Html element by using Pixel(px), Percent(%), Points(pt), and Inches(in).
