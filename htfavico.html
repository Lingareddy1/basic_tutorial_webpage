HTML Favicons

  • Favicons are also referred as favourite icons,which generally appears in the header section of the browser.

  • Favicons will appear just before the title of the tab.The combination of favicon and title is very important for the header section of the browser.

  • Link tag is used to provide the favicons on the browser.It takes two important attributes they are,

    1. "rel" attribute : stands for relation ,which considers the type of the file which has to be link.

    2. "href" attribute : stands for hyperlink reference ,which takes the icon file filename along with ".ico" extension.

  • Link tag must be used under the head tag.

  • Note : File Population is very important to get the expected result.It means grouping of all the necessary files under one folder.
