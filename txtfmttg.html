Text Formatting Tags

  • Programmers will make use of text formatting tags to change the apperance of the given text contents.Few commonly used text formatting tags are,

    1. Strong Tag : It is used to make the contents to appear as bold .


    2. Emphasize Tag : It is used to make the text contents to appear as Italic . it is represented by "em" tag where 'em' stands for emphasize.


    3. Mark Tag: It is used to highlight the given text content by providing the background color yellow as default.


    4. h1 to h6 Tags : It includes h1, h2, h3, h4, h5 and h6 tags. "h1" tag provides the highest font size in Html whereas "h6" tag provides the lowest font size in Html. h1 to h6 tags font size decreases gradually.

      Note : h1 to h6 tags can move contents to the next line without using the break tag.

    5. Superscript Tag : It is generally provides the contents which will appears above the text.It is is represented by sup tag.


    6. Subscript Tag : It is generally provides the contents which will appears below the text.It is is represented by sub tag.


    7. Insertion Tag : It is used to provide underline for the given text contents.It is is represented by ins tag.


    8. Deletion Tag : It is used to provide strike on the given text contents.It is is represented by del tag.


    9. Marquee Tag : It is used to provide scrollings on the browser.It takes three important attributes they are

      1. Direction : It is used to control the direction of scrollings has to move.

        1. Right

        2. Left (Default)

        3. Up

        4. Down

      2. Scrollamount: It is used to control the speed of the scrolling.Higher the value faster the speed of the scrolling.

      3. Loop : It is used to control the number of times the scrolling has to appear on the browser.

  • Note : Marquee tag is referred as deprecated tag,which means it is outdated.But in case if it is used we get the expected result.

  • Note : Background colour given by mark tag can be changed by using "style"   attribute.This attribute can be used with any HTML tags.
