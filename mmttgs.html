Multimedia Tags

  • Programmers prefer multimedia tags in order to provide advertisements on the browser.

    1. Image Tag : It is used to provide images on the browser.It takes four attributes they are,

      1. src : It stands for source attribute which is basically used to provide image filename along with extension.

      2. Width & Height : These are used to control the dimensions of the image.

      3. alt : It stands for "alternate text" used to give some kind of message if given file does not exist.

    2. Audio and Video Tag : It is used to stream the audio and video files on the browser.The common attribute between these tags are " src " & " controls " is used to display play,pause buttons for the audio and video files.

    3. Note : As a programmer we can change the dimensions of video and image by using " width " &          " height " attribute.

  • Center tag is used to move the image or any file to the center of the Html page on the browser.
