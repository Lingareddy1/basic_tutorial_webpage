CSS Margin

  • it is used to create the spaces around Html elements but outside the border.

  • Programmers can apply margin to all the sides as well as specific side of the element.

  • To apply the margin for the specific side of the element, we have to make use of following properties

    1. Margin-left

    2. Margin-right

    3. Margin-top

    4. Margin-bottom
