CSS Padding

  • It is used to create the spaces around Html elements or text contents but inside the border.

  • It can be apply margin to all the four sides as well as specific side of the element.

  • To apply the padding for the specific side of the element, we have to make use of following properties

    1. padding-left

    2. padding-right

    3. padding-top

    4. padding-bottom
