CSS Icon Library

  • In CSS all the icons which are necessary are stored under "CSS Icon Library".

  • To use the icon from the Icon library,programmers have to follow the few steps,

    1. Attach the Html file with CSS Icon library by using 'link' tag.
    2. Browse fontawesome.com website and search for appropriate icon required.

    3. Copy 'i' tag and paste it into the Html file.


    4. In Html5 'i' tag is used to represent CSS icon.

  • Note : To change the colour of icon use colour CSS property and to change the size use fa-1x, 2x, 3x,.....10x.
